Security Hardware

Protecting everything physical.

TPM

Token Package Manager - a small piece of hardware that integrates into the motherboard. Its purpose is to store all your encryption keys as well as the BIOS settings.

Cable Lock

A durable steel threaded cable that locks onto critical equipment to secure it and prevent theft..

Surveilence Cameras

The most common type of security infrastructure, allows us to keep track of infrastructure usage and ensure authorised personel.

Boulards

Poles that stick out of the ground to stop heavy vehicles from entering an area.

Access Control Vestibules

Infrastructure usually located by the entrance of a building, helps prevent tailgating and ensures only authorized personel may enter.


Security Software

Protecting your system from cyberthreats such as malware, phishing, viruses etc.

Antivirus Software

Scans your system for any threats or malicious activity and cleans any viruses.

VMware

A virtual instance of an Operating System where you can run multiple Operating systems simultaneously, furthermore you can test any software in a virtual environment without effecting the host computer if anything goes wrong.

Password Manager

Stores all of your passwords in a protected environment.

PC Health

Keeps you updated on your PC updates and any issues that can lead to vulnerabilities.


Network Security

Ensure data packets arent tampered with during transmissions.

VPN

Secures your connection to remote servers via an encrypted tunnel for data transmissions.

DMARC

Creates a backup plan incase your spam gateway and DKIM are compromised.

Server Monitoring

ManageEngine and Zibbix are some popular options for software you can install on your computer to monitor and managae servers and their performance.

Firewalls

A network gateway application that lets you control incoming and outgoing traffic.

WPA2

Encryption standards.
